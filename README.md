## 📦 Projetos em Destaque

### 1️⃣ **Farmácia Escola - Sistema Clínico (TCC Aprovado)**
**Trabalho de Conclusão de Curso | Jun/2025**

Plataforma completa de gestão clínica transformou operação manual de uma clínica-escola em sistema estruturado.

- **Problema Real**: Farmácia operava com planilhas e fichas físicas
- **Solução**: Sistema Django/PostgreSQL com cadastro de pacientes, consultas, anamneses, dashboards operacionais
- **Impacto**: Centraliza 100+ pacientes/mês, gera relatórios automatizados, fundação para pesquisas científicas
- **Stack**: Python, Django, PostgreSQL, JavaScript, Chart.js, Design Centrado no Usuário
- **Status**: ✅ Aprovado com conceito máximo | Pronto para produção

### 2️⃣ **Portal DECOM - Sistema Institucional**

Portal web institucional com autenticação segura, landing page editável e gerenciamento centralizado de convites.

- **Problema**: Instituição precisava de portal com conteúdo público editável e acesso seguro apenas para membros
- **Solução**: Django com separação clara: landing pública (editável), área autenticada, painel administrativo
- **Destaques**: Autenticação por email, fluxo de convites com tokens, painel de gestão sem acesso ao BD
- **Stack**: Django, Python, HTML5, CSS3, SQLite, Django Allauth

🔗 **Repositório**: [github.com/EnicarlosJr/Decom](https://github.com/EnicarlosJr/Decom)

---

### 3️⃣ **CMS para Artigos - Content Management System**

Gerenciador de conteúdo para publicação de artigos com categorização, busca e painel administrativo intuitivo.

- **Problema**: Publicar conteúdo sem estrutura gera dependência de desenvolvimento e falta de padronização
- **Solução**: CMS simples com CRUD, categorização, busca, admin customizado
- **Destaques**: Fluxo editorial independente, organização por categorias/tags, fundação escalável
- **Stack**: Python, Django, PostgreSQL, Bootstrap, JavaScript

🔗 **Repositório**: [github.com/EnicarlosJr/cms-artigos](https://github.com/EnicarlosJr/cms-artigos)
